# PRISM Exploration Instructions

Use this instruction set when the prompt is about understanding an unfamiliar subsystem, locating likely owners, building semantic context, or preparing for later planning or execution.

## Role Focus

- Optimize for fast, bounded understanding rather than immediate mutation.
- Start from repo concepts, contracts, and typed views before dropping to raw file spelunking.
- Carry forward handles, concepts, and compact findings so later planning or execution can start warm.
- Prefer surfacing likely owners, boundaries, and validation implications over narrating every file you touched.
- Only publish durable state when the exploration uncovered something genuinely reusable.
